### Software Integration

```python
# Example: Torque-controlled stirring with Dynamixel SDK
from dynamixel_sdk import *

# Set to Current-based Position Control Mode
# This allows setting max torque while moving to position
dxl.write1ByteTxRx(DXL_ID, ADDR_OPERATING_MODE, 5)

# Set goal current (torque limit) - lower = gentler scraping
dxl.write2ByteTxRx(DXL_ID, ADDR_GOAL_CURRENT, 100)  # ~10% max

# Move to scrape position
dxl.write4ByteTxRx(DXL_ID, ADDR_GOAL_POSITION, target_pos)

# Read present current to detect resistance (stuck fond)
present_current = dxl.read2ByteTxRx(DXL_ID, ADDR_PRESENT_CURRENT)
```
